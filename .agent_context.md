# General Chemistry II Research Project – Agent Context

**Project:** Catalyst Insight Studio – a Streamlit web app for chemistry professors to track student trends using ML.

**Structure:** Two main Python modules (~2500 lines total):
- `app.py` (918 lines): Streamlit UI, page layout, widgets, calls to pipeline
- `pipeline.py` (1551 lines): Data loading, cleaning, encoding, ML training, visualization, statistical analysis

**Stack:** Python 3.12, Streamlit >=1.32, pandas >=2.0, scikit-learn >=1.3, numpy >=1.24, matplotlib >=3.7

**Core Workflow:** Load spreadsheet data → Clean & encode → Add chemical descriptors → Build feature matrix → Train ML model (regression/classification) → Generate plots & predictions

**Key Modules in pipeline.py:** Functions for data validation, chemical descriptor computation, ML model training, visualization (bar charts, heatmaps, distributions), and export utilities.
